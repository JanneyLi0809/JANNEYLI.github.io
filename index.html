How to Build a Personal Website (Hexo+Github)

2019-12-14

Welcome to Janney Li’s blog! This is my first post. I’m gonna record how did I built my personal website. or you can ask me on GitHub.

Let’s Start

Tools

HARDWARE:
Computer(win10)
Internet

SOFTWARE:
Node.js
Git

All the options encountered in the download follow the recommendations

Join Github

  1. CLICK https://github.com UPPER RIGHT CORNER sign up

  2. CREATE Repository

    Repository Name: username.github.io EX: JANNEYLI.github.io

Set Up SSH keys

  2. Generate a new SSH Key:
    SSH Keys is used to contact the local git project with GitHub, so that the blog project on GitHub is the latest update.

    Enter the email address you Github Username
    You will then be asked to enter your password:

Finally seeing this, you successfully set the ssh key:

  1. Add SSH Key to GitHub

Find the id_rsa.pub file in your local folder, they usually under your USER folder.

Open the file in Notepad in the .ssh folder and COPY the entire content to the corresponding location on github
Back on your GitHub homepage, click on the avatar in the upper right corner and select Setting

Continue selecting SSH and GPG keys in the left menu bar

Paste Key, click Add SSH key

Test

Success! Next step!

Set up a Hexo blog

Install hexo with npm command

Enter the hexo init initialization folder,

Enter npm install to install the necessary components.

Enter hexo g to generate a static web page,

Enter hexo s to open the local server, and then open http: // localhost: 4000 / in your browser. You can see our blog. The effect is as follows:
